## Workflow

1. **Read** a paper → create a note in `literature/`
2. **Design** an experiment → create a file in `experiments/` with status `planned`
3. **Run** the experiment → update status to `running`, record observations
4. **Analyze** → update status to `completed`, record results
5. **Synthesize** → connect findings across sources in `notes/`
