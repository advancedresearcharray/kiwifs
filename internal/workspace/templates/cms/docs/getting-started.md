# Getting Started

Welcome! This guide will help you get up and running quickly.

## Prerequisites

- An account (sign up at example.com)
- A modern web browser

## Quick Setup

1. **Sign in** to your dashboard.
2. **Create a workspace** — give it a name and invite your team.
3. **Start creating** — use templates or start from scratch.
