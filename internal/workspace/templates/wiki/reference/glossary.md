# Glossary

Shared vocabulary for the team. Add terms alphabetically.
When you use a term that might be unfamiliar, link here:
`[[reference/glossary|Glossary]]`.

## A

**ADR** — Architecture Decision Record. A short document capturing
a significant decision, its context, alternatives, and rationale.
See [[decisions/index|Decisions]].

## D

**Deploy** — Ship code from a branch to an environment. See
[[processes/deployment|Deployment Process]].

## S

**SOP** — Standard Operating Procedure. A step-by-step guide for
a recurring task. Lives under [[processes/index|Processes]].

**Standup** — Daily 15-minute sync to surface blockers and handoffs.
See [[how-we-work]].
